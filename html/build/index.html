Web-Serve

A High Performance Production Build Server mainly focused on serving SPA(Single Page Applications).

  • Written in Golang. Lightweight, Blazing Fast.
  • Embeds all static assets(build assets) into a single binary with the server. No need to carry around `build` directory.
  • Supports running as a Platform Agnostic Daemon/Service. No overhead of killing the process by mistake.

The Project has been released as open source. Feel free to contribute ^_^
